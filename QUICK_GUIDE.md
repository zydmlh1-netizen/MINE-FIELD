# ⚡ دليل البدء السريع - اللعب مع لاعب حقيقي

## 🚀 5 خطوات سهلة

### ✅ الخطوة 1: تحضير Firebase (مرة واحدة فقط)
```
1. اذهب إلى https://console.firebase.google.com
2. أنشئ مشروع جديد
3. فعّل Realtime Database
4. انسخ البيانات (firebaseConfig)
5. أرجع للعبة وحدّث firebase-config.js
```
📖 **تفاصيل أكثر**: انظر SETUP.md

---

## 🎮 أثناء اللعبة

### اللاعب رقم 1️⃣ (المضيف)
```
1. انقر "ابدأ المعركة" 🔥
2. انقر "إنشاء غرفة" 🏰
3. اختر الإعدادات (حجم الشبكة والألغام)
4. انقر "تحدى لاعب حقيقي" 👥
5. اضغط "إنشاء"
6. ❗ انسخ الكود (مثال: ABCD12)
7. أرسل الكود للاعب الثاني
```

### اللاعب رقم 2️⃣ (الضيف)
```
1. انقر "ابدأ المعركة" 🔥
2. انقر "انضمام" 👥
3. الصق الكود (مثال: ABCD12)
4. اضغط "انضمام"
5. ⏳ انتظر التحقق (3 ثواني)
6. ✅ إذا نجح: ستسمع صوت الانضمام
```

## ✅ إذا عملت بدون مشاكل:
```
✅ كلاهما في نفس الغرفة
✅ المضيف ضع 10 ألغام في 20 ثانية
✅ الضيف ضع 10 ألغام في 20 ثانية
✅ ابدأ اللعبة - أدوار متبادلة
✅ آخر من ينفجر لغمه = الخاسر
```

## ❌ إذا ظهر خطأ

### الخطأ: "لم يُعثر على الغرفة"

**التحقق السريع:**

| ماذا تفعل | النتيجة | الحل |
|---------|--------|------|
| هل الكود 6 أحرف؟ | نعم ✅ | تابع التحقق |
| هل المضيف أنشأ أولاً؟ | نعم ✅ | تابع التحقق |
| هل Firebase متصل؟ | ؟ | اضغط F12 و"انقر Console |
| هل Firebase صحيح؟ | ؟ | انظر SETUP.md |
| هل لديك إنترنت؟ | نعم ✅ | تابع التحقق |

**حل سريع:**
```
1. أغلق المتصفح تماماً
2. افتح متصفح جديد
3. جرب مرة أخرى
```

📖 **حل شامل**: انظر TROUBLESHOOTING.md

## 🎯 الأوضاع المتاحة

### 🤖 ضد الذكاء الاصطناعي
- ✅ يعمل دائماً (بدون Firebase)
- ✅ سريع وسهل
- ✅ اختبر قواعد اللعبة

### 👥 ضد لاعب حقيقي
- ⚠️ يحتاج Firebase محكم
- ⚠️ نفس الشبكة أو إنترنت
- ✅ تحدي مع أصدقائك

### 📱 نفس الجهاز (نافذتي متصفح)
- ✅ يعمل دائماً
- ✅ اختبار ممتاز
- ✅ لا تحتاج إنترنت
